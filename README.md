# Context Engineering

**The Discipline Behind Effective AI Software Development**

By Alexander Mikhalev

## About This Book

This repository contains the source for *Context Engineering*, a technical book exploring how to build reliable AI software through disciplined context management, knowledge graphs, and token budget architecture.

## Building

Requires [Quarto](https://quarto.org/):

```bash
quarto render
```

Output appears in `_book/`.

## Structure

- `chapters/` — 22 chapters in Quarto markdown (.qmd)
- `appendix/` — 7 appendices
- `_quarto.yaml` — Book configuration
